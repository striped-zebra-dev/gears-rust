# Get an unknown consumer group

Looking up an id that was never registered (or was deleted) returns `404`.

## Request

```http
GET /v1/consumer-groups/gts.cf.core.events.consumer_group.v1~00000000-0000-0000-0000-000000000000 HTTP/1.1
Host: broker.example.com
Authorization: Bearer <tenant-token>
```

## Expected response

- `404 Not Found` (`PD`)

```json
{
  "type": "gts://gts.cf.core.errors.err.v1~cf.core.err.not_found.v1~",
  "title": "Not Found",
  "status": 404,
  "detail": "consumer group gts.cf.core.events.consumer_group.v1~00000000-0000-0000-0000-000000000000 not found",
  "instance": "/v1/consumer-groups/gts.cf.core.events.consumer_group.v1~00000000-0000-0000-0000-000000000000",
  "trace_id": "<request-trace-id>",
  "context": {
    "resource_type": "gts.cf.core.events.consumer_group.v1~",
    "resource_name": "gts.cf.core.events.consumer_group.v1~00000000-0000-0000-0000-000000000000"
  }
}
```
