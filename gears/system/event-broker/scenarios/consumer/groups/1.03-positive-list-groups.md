# List consumer groups visible to the caller

Read-only listing of groups in the caller's tenant.

## Request

```http
GET /v1/consumer-groups HTTP/1.1
Host: broker.example.com
Authorization: Bearer <tenant-token>
```

## Expected response

- `200 OK`
- Body is an array of group records the caller's principal may see (tenant-scoped).

```json
[
  {
    "id": "gts.cf.core.events.consumer_group.v1~7b2c1e9a-4f3d-4a2b-9c8e-1d2f3a4b5c6d",
    "tenant_id": "<tenant-uuid>",
    "owner_principal_id": "<principal>",
    "kind": "anonymous",
    "description": "order-fulfilment workers",
    "created_at": "2026-05-29T10:00:00Z"
  }
]
```
